# Prácticas de algoritmos con PSeINT

## Descripción
> Este repositorio contiene la resolución de ejercicios prácticos correspondientes al curso "Lógica de Programación" impartido por Sergio Alejandro Pérez Gabriel en Udemy. El objetivo principal es consolidar los fundamentos de la programación y el pensamiento algorítmico.
>
> **🚀 Objetivos del Proyecto**
> * Dominio de la herramienta: Implementación de soluciones utilizando PSeInt.
> * Estructuras de Control: Aplicación de condicionales, bucles (ciclos) y estructuras secuenciales.
> * Pensamiento Lógico: Resolución de problemas técnicos mediante el diseño de algoritmos eficientes en pseudocódigo.
>

## Instalación y configuración 🚀
<b>1️⃣ Requisitos</b>
* PSeInt instalado

<br>

<b>2️⃣ Instalación de PSeInt</b>
1. Descargar PSeInt desde: https://pseint.sourceforge.net
2. Instalar siguiendo las instrucciones del sistema operativo.
3. Abrir PSeInt.

<br>

<b>3️⃣ Cómo descargar el repositorio </b>
<br>
No necesitas instalar nada complicado. Puedes obtener el repositorio de dos formas:
1. Clonar el repositorio con Git (Si cuentas con experiencia): `git clone https://github.com/odrasanchezdev/PSeInt-practica`
2. Descargar el ZIP desde GitHub: En esta página principal, busca el botón azul `Code` y despliega el menú. Da clic en el botón“Download ZIP” y descomprímelo en tu equipo.
3. No te preocupes, puedes copiar y pegar directamente el ejercicio en tu editor. Si recién incias tu camino en la programación.

<br>

<b>4️⃣ Ejecutar archivos .psc descargados</b>
1. Abrir PSeInt.
2. Seleccionar Archivo → Abrir.
3. Elegir el archivo .psc de tu interés.
4. Presionar Ejecutar (o F9).

<br>

<b>4️⃣.1️⃣ Ejecutar archivos .psc nuevos</b>
<br>
Con PSeInt abierto.
1. Seleccionar Archivo → Nuevo.
2. Pegar el código del archivo de tu interés desde el repositorio.
4. Presionar Ejecutar (o F9).
    * En la barra lateral derecha, tienes la opción de *Ejecutar paso a paso*, así, puedes observar en pantalla un indicador sobre la instrucción que se está ejecutando en ese momento.
